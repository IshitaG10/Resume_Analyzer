<p>1. Install virtual enviornment</p>


```
virtualenv 'name'
```

<p>3. Activate the virtual enviornment</p>

```
source venv/bin/activate
```

<p>4. Install the packages</p>

```

```


<p>6. Copy the format from</p>

```

```

```
streamlit run app.py
```
